# reecewal.sh

The source code for my personal website. This project currently uses Next.js, tailwindcss, and Three.js.

## Getting Started

Run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
